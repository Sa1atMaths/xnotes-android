xnotes is a handwriting-first notebook for phones and tablets, built for pen and stylus input. Write, sketch and annotate with pressure-sensitive, variable-width ink, then organise your work across multiple pages.

Features:

* Pressure-sensitive ink with pen and highlighter tools
* Shapes, text boxes and image insertion
* Stroke and object erasers, plus box and lasso selection with move/resize
* Multi-page documents with several page sizes
* Import and export PDF files
* Save and open notebooks in the .xnote format
* A backstage file area: recent notes, a built-in folder explorer and quick sharing
* Undo and redo
* Side panel with page thumbnails, contents and bookmarks
* Stylus-aware input: finger panning, pinch to zoom and a configurable pen side-button action
* A live presentation mode that streams the canvas over your local network
* Customisable theme, accent colour and page defaults
